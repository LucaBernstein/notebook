# Pronounciation

## Past Tense

#### `did` / `dıd`

When the word ends with `t` or `d`, pronounce it as `dıd`:

* `started` -> `star/dıd/`
* `invited` -> `invite/dıd`
* `expected` -> `expec/did`

* `ended` -> `en/did`
* `decided` -> `deci/dıd`
* `needed` -> `nee/did`

#### `t`

* `walked` -> `waakt`
* `talked` -> `taakt` 
* `watched` -> `watch/t/` (voiceless)
* `asked` -> `askt` 
* `shopped` -> `shopp/t/`
* `coughed` -> `cough/t/`

#### `d`

* `rained` -> `rain/d/`
* `showed` -> `show/d/`
* `planned` -> `plann/d/`

#### `ıd`

* `demanded` -> `deman/ded`
* `waited` -> `wai/ded`
* `wanted` -> `wan/ded`
